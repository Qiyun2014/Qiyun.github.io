CMakeLists编写

单文件编译

  • cmake_minimum_required : 指定配置文件的CMake最低版本；
  • project : 标识项目名称
  • add_executable : 将名为xxx.cc的源文件编译成一个设置好的项目名称的可执行文件

多文件编译

单个目录，多个文件

CMake 将当前目录所有源文件赋值给变量DIR_SRCS，再指定DIR_SRCS中的所有源文件编译成一个myApplication的可执行文件
